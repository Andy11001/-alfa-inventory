# Skrypty badawcze / jednorazowe

Narzędzia użyte przy budowie feedów (głównie odkrywanie struktur DS/Alfa:
kolory, LCDV, konfigurator). **Żaden workflow ich nie uruchamia** — to
archiwum warsztatowe. Mogą wymagać `PYTHONPATH` wskazującego na `scrapers/`
i nieaktualnych plików w `data/`.

Produkcyjne scrapery żyją piętro wyżej w `scrapers/`.
